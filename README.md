# 🧾 InsightMail

### _From raw data to smart reports, automatically._

---

## 📌 About the Project

InsightMail is a full-stack web application that allows users to
upload CSV data, visualize it as charts, and automatically send
a professionally formatted PDF report to any email address —
powered by a Java Spring Boot backend.

---

## ✨ Features

- 📂 Upload CSV files and preview data instantly
- 📊 Auto-generate Bar, Pie, and Line charts (Chart.js)
- 📄 Generate professional PDF reports (iText - Java)
- 📧 Send PDF reports via email (JavaMail + SMTP)
- 🗂️ View report history with timestamps
- 🔐 User login and signup

---
